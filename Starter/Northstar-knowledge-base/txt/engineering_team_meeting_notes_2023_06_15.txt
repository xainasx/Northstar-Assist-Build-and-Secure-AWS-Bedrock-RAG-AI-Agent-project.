3. AI ASSISTANT PROJECT UPDATE (Northstar Assist)

James: We're moving forward with the RAG architecture using Amazon Bedrock. Current plan:
- Foundation model: Claude 3.7 Sonnet (via Bedrock)
- Embedding model: Amazon Titan Text Embeddings V2
- Vector store: Amazon OpenSearch Serverless
- Knowledge source: Internal documents stored in S3

Dev: I want to flag early that we need a proper security review before this goes live.

James: Agreed. Let's plan a threat modelling session for next sprint.

Marcus: I'll put Dev and James together to lead that. Target: security review complete
